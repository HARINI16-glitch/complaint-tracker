## üìå About the Website

The Complaint & Issue Management System allows users to submit complaints easily without any login.  
Administrators can securely log in to access a dashboard where they can view, filter, prioritize, and resolve complaints.

The system separates **public access** and **admin access**, ensuring better control and real-world workflow simulation.
